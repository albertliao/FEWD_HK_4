## The clearfix hack
<aside class="notes">Mart ~ 10 mins : @FEWD_HK_4/lesson3/labs/codepen-clearfix</aside>

Apply the clearfix class to the parent element such that it knows the height of the floating children.

```css
/* Clear fix hack */
.clearfix:after {
  content: ".";
  display: block;
  clear: both;
  visibility: hidden;
  line-height: 0;
  height: 0;
}
```
